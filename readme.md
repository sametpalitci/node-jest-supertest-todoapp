## Getting Started

Remember this project is made for learning!

Follow the steps!

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sametpalitci/node-jest-supertest-todoapp.git
   ```
2. Install Packages
   ```sh
   yarn
   ```
3. Convert to .env.sample to .env
4. Enter your enviroment info
5. It's Done.
